Langkah2 :

1. Install melalui composer di laravel.com
2. Nyalakan server dan my sql nya
3. Install extension laravel di vscode
4. Edit .env untuk menyesuaikan database
5. php artisan make:model
6. php artisan migrate (untuk membuat tabel di database) 
7. cek di database apakahh tabel sudah ada
